## Overview

gentleduck/iam ships skills for AI coding agents. Skills follow the [Agent Skills specification](https://agentskills.io/specification) and work with Claude Code, GitHub Copilot, Cursor, Cline, OpenCode, and 30+ other agents.

## Install

```bash
npx skills add gentleeduck/duck-iam
```

## Available Skills

| Skill | Package | What it covers |
|---|---|---|
| `duck-iam` | `@gentleduck/iam` | RBAC and ABAC authorization, role definitions, policy composition, middleware, React client |

## MCP Documentation Server

The docs site exposes an MCP server:

```
https://iam.gentleduck.org/api/mcp
```

To connect your agent, add this to your project's `.mcp.json`:

```json
{
  "mcpServers": {
    "duck-iam-docs": {
      "type": "url",
      "url": "https://iam.gentleduck.org/api/mcp"
    }
  }
}
```

## How Skills Work

Each skill is a `SKILL.md` file with YAML frontmatter and markdown instructions. The agent reads the `description` to decide when to activate the skill. The markdown body provides the instructions. Reference files in a `references/` subdirectory provide deeper context loaded on demand.
