## duck-gen <Badge>feat</Badge>

Built initial implementation of **duck-gen** -- a code generation tool that scans server code and emits TypeScript definitions for API routes and message keys.

- Configuration schema with JSON validation
- CLI for running generation from the terminal
- API route scanning from NestJS controller decorators
- Message scanning from `@duckgen` JSDoc tags
- TypeScript `.d.ts` output with zero runtime cost
- Added LICENSE files and fixed README formatting

## duck-query <Badge>feat</Badge>

Built initial implementation of **Duck Query** -- a type-safe Axios HTTP client that uses route metadata from Duck Gen.

- Type-safe API client with full TypeScript inference
- Request shape validation (body, query, params, headers)
- `CleanupNever` utility to strip inapplicable fields
- Seven typed client methods (`get`, `post`, `put`, `patch`, `del`, `request`, `byMethod`)
- Added Duck Query documentation, FAQs, and templates
