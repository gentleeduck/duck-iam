## Monorepo Split <Badge>breaking</Badge>

Moved `duck-gen`, `duck-query`, `duck-skitch`, and `duck-ttest` packages out of the `@duck-ui` monorepo into the standalone `@duck-gen` monorepo.

- Separated duck-gen ecosystem into its own repository at [`@duck-gen`](https://github.com/gentleeduck/duck-gen)
- Each package retains its full git history
- Updated all internal import paths and workspace references
- Migrated from pnpm to **Bun** as the package manager

### Packages moved

| Package | Description |
| --- | --- |
| `@gentleduck/gen` | Code generation CLI and type emitter |
| `@gentleduck/query` | Type-safe Axios HTTP client |
| `@gentleduck/skitch` | Experimental interpreter package |
| `@gentleduck/ttest` | Type-level testing framework |
| `duck-gen-docs` | Documentation site |
